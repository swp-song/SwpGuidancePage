### 导入

> * 手动导入：
> 
>> * **SwpGuidancePage** 文件夹导入项目中。
>> * **`#import "SwpGuidancePageHeader.h`**
>> 
> -------

> * CocoaPods 导入:
> 
>> * **pod search SwpGuidancePage**
>> * **pod 'SwpGuidancePage'**
>> * **`#import <SwpGuidancePage/SwpGuidancePageHeader.h>`**
>>
